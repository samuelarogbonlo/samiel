# samiel

Personal website for Samuel Arogbonlo at [samiel.me](https://samiel.me).

This is a small Jekyll site inspired by old-school single-column personal pages.

## Local development

```sh
bundle install
bundle exec jekyll serve
```

The site will be available at `http://127.0.0.1:4000`.

## Editing

- Homepage: `index.md`
- Contact page: `contact.md`
- Blog index: `blog.md`
- Posts: `_posts/`
- Navigation: `_data/nav.yml`
- Site config: `_config.yml`
- Styles: `_sass/`
